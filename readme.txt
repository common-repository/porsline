== Description ==

<h3 style="text-decoration: underline;">Before Using Porlsine plugin:</h3>
<p>First, simply register in Porsline. It is free, no credit card is required.  
After that, install the plugin. Each user of Porsline has an API key. With the help of plugins and using the API key, you can call the forms and surveys that you have created in Porsline and embed them on every page of your WordPress website.</p>

<p>With a simple drag & drop and installation of Porsline WordPress plugin, you can benefit from these features:</p>
<ul>
    <li>Use ready-made, fully customized form templates.</li>
    <li>Apply your desired design or your company brand identity to the form.</li>
    <li>Create an unlimited number of forms.</li>
    <li>Add Logic to personalize and segment your audience.</li>
    <li>Gather unlimited responses.</li>
    <li>Add score, set answer key, and auto-grading to create engaging quizzes.</li>
    <li>Randomize questions & choices in online quizzes.</li>
    <li>Show people customized endings based on their answers</li>
    <li>Export responses as Excel-supported formats.</li>
    <li>Real-time interactive statistical reports and chats.</li>
    <li>Slice & dice your data to make customized reports.</li>
    <li>Create multilingual forms: Arabic, Turkish, Spanish, Russian, English, and Persian.</li>
    <li>Set auto-start, auto-terminate, and response duration of the forms.</li>
    <li>Authenticate respondents via email, code, and OTP.</li>
    <li>Use RESTful API to seamlessly integrate with external software.</li>
</ul>

== Frequently Asked Questions ==

= How to create a form? =

Making glorious forms is only a simple drag & drop away! Just log in to your account, select Create new survey or pick from more than 300 templates. Drag & drop question types and write down your questions. 
Watch "<a href="https://porsline.com/videos#QmFCD" target="_blank">How Porsline Works?</a>" 

= What kinds of forms & surveys can I create? =

Using Porsline you can create <a href="https://porsline.com/survey-templates/contact-form-template" target="_blank">contact forms</a>, surveys, lead generation forms, <a href="https://porsline.com/survey-templates/order-form-template" target="_blank">order forms</a>, <a href="https://porsline.com/survey-templates/event-registration-form" target="_blank">event registration forms</a>, <a href="https://porsline.com/survey-templates/registration-form-template" target="_blank">signup forms</a>, <a href="https://porsline.com/survey-templates/feedback-form-template" target="_blank">feedback forms</a>, booking forms, polls, petition forms, upload forms and appointment forms. There are over 300 <a hraf="https://porsline.com/survey-templates/" target="_blank">ready-to-use form templates</a> on Porsline. You can find one and save time creating your form.

= Is it free? =

Porsline have a free forever plan. Porsline is free up to 100 form submissions per month. If you go over that limit and also need more advanced features you will need to upgrade to a paid plan.
